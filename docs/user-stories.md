# Clock Keeper — User Stories

All stories are from the Storyteller's perspective. MVP covers Game Setup, Night Management, and Note-Taking & Tracking.

## Game Setup

- Start a game from a saved script or create a new one
- Create a new script: pick characters from the full character list, set player count — app validates correct distribution (Townsfolk / Outsider / Minion / Demon counts)
- Randomize role selection: given a script and player count, randomly select which roles are in play
- Manually adjust roles: swap, add, or remove roles from the randomized selection
- See which reminder tokens are needed based on the selected characters
- Save a script for reuse in future games
- Follow a step-by-step setup checklist (based on the official rules setup sheet) guiding the Storyteller through everything before the game starts
- Role assignment to players happens offline (physical tokens) — the app only tracks which roles are in play, not who has them

## Night Management

- See the night order for the current script, filtered to only living/relevant characters
- Step through the wake sequence one character at a time
- See an ability prompt for each waking character (what to say, show, or ask)
- Record night actions as they happen (Demon pick, Poisoner target, info given to Empath, etc.)
- Distinguish first night vs other nights (different wake orders)
- See which reminder tokens to place or move when resolving each character's ability
- Mark a character as done and auto-advance to the next

## Note-Taking & Tracking

- Seating chart showing all players with alive/dead status
- Mark players as dead (with or without ghost vote remaining)
- Record nominations: who nominated whom, vote count, passed/failed
- Record executions
- Track game-altering state: poisoned, drunk, protected, ability used
- Per-phase timeline (Night 1 → Day 1 → Night 2 → …) with notes attached to each phase

## Player Reference *(future)*

- Look up any character's ability text
- See jinx interactions for the current script
- View script summary with all characters grouped by type

## AI-Assisted Features *(future)*

- Setup recommendations (balanced composition for player count)
- Night phase hints (e.g. "Empath gets 0 — no evil neighbors alive")
- Post-game analysis (timeline review, key moments)
